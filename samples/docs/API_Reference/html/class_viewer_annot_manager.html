◆ setViewerAnnotFlag()

ViewerAnnotManager::setViewerAnnotFlag (   getAnnotFlagValue)
inline

Set annotation flags for PDFViewer.

Parameters
getAnnotFlagValue{Function} This return value merges with the flags of the PDF annotation itself
Example:
function example (annotManager) {
annotManager.setViewerAnnotFlag(function (annot) {
var AnnotFlagEnum = {
invisible: 1,
hidden: 2,
print: 4,
noZoom: 8,
noRotate: 16,
noView: 32,
readOnly: 64,
locked: 128,
toggleNoView: 256,
lockedContents: 512,
};
if (annot.getTitle() === 'FoxitPDFSDKForWeb') {
return AnnotFlagEnum.readOnly | AnnotFlagEnum.locked;
}
return 0; // all flags
})
}
