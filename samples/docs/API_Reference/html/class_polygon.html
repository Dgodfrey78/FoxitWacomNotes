Polygon::setFillColor (   fillColor)
inline

Set annotation's fill color. [Support in Server]

Parameters
fillColor{*} - An interge indecated fill color (alpha is not supported). Supported color format:
  • 0 or ['T'] for transparent
  • number, such as 0xff0000, 16711680
  • PDF color array, such as ['RGB',1,1,1] which means white.
  • HTML color format, such as '#ff0000', 'rgb(255,0,0)', 'rgba(255,0,0,1)'
Returns
{Promise.<boolean>} - A boolean value indecated success or failue.
